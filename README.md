# Ziplark Homebrew tap

Free, fast, cross-platform archiver — [ziplark.com](https://ziplark.com) ·
[source](https://github.com/zhitongblog/ziplark).

## Install

Desktop app:

```bash
brew install --cask zhitongblog/tap/ziplark
```

CLI + MCP server (`ziplark`, `ziplark-mcp`):

```bash
brew install zhitongblog/tap/ziplark
```
